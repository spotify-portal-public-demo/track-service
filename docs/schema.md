# BigQuery Schema for Track Dataset

```
+----------------------------------+
|              Schema              |
+----------------------------------+
| |- id: string (required)         |
| |- isrc: string (required)       |
| |- title: string (required)      |
| |- duration: integer (required)  |
| |- album_id: string (required)   |
| |- artist_ids: string (repeated) |
+----------------------------------+
```
